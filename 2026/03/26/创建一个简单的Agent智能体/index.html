首先，确保你的电脑中配置了Python环境（需要Python版本3.10+），并安装以下两款开发者工具：

uv (极速 Python 包与环境管理器)

  • 官方文档与指南主页: https://docs.astral.sh/uv/

  • GitHub 仓库: https://github.com/astral-sh/uv

  • 如果没有安装，请通过命令行安装，打开你的电脑终端（Windows 是 PowerShell，Mac 叫 终端/Terminal），复制并运行以下对应的命令：

  • 如果你用的是 Windows (在 PowerShell 中运行):

    PowerShell

  • 如果你用的是 macOS 或 Linux:

    Bash

(安装完成后，可能需要重启一下终端，输入 uv --version，如果出现版本号就说明安装成功了。)

Jupyter Notebook (交互式代码编写神器)

  • 官方主页: https://jupyter.org/

由于我们开发的是智能体项目，所以我们需要申请主流大模型的API密钥。这里以阿里的通义千问(QWen)为例：
1.注册与获取API密钥
访问 阿里云百炼平台 (bailian.aliyun.com)。

  • 登录后，进入工作台。
  • 在左侧导航栏找到并点击 “API-KEY 管理”。
  • 点击 “创建新的 API-KEY”，复制生成的那串以 sk- 开头的字符串。
  • LangChain 默认识别的环境变量名：DASHSCOPE_API_KEY
